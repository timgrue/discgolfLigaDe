# Berliner Discgolf Liga

A Vue 3 + TypeScript website for the Berliner Discgolf Liga (Berlin Disc Golf League).

## Features

- **Team Standings**: View current league standings with wins, draws, losses, and points
- **Match Schedule**: See upcoming matches with dates, times, and course locations
- **Course Spotlight**: Detailed information about the featured disc golf course where the next match will be played

## Tech Stack

- Vue 3 with Composition API
- TypeScript
- Vue Router
- Vite
- ESLint + Prettier

## Getting Started

### Prerequisites

- Node.js >= 20.19.0

### Installation

```bash
cd frontend
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Linting

```bash
npm run lint
```

### Type Checking

```bash
npm run type-check
```
